OVERVIEW

Challenge: Design a website that that provides a platform to showcase small businesses and hidden gems often overlooked by mainstream travel guides

Opportunity: Create an engaging platform that connects travelers with the rich stories behind LA’s small businesses. This initiative supports local economies while inspiring visitors to explore the city’s authentic culture and lesser-known attractions.

Technologies: UI/UX | HTML | CSS | JavaScript

Frameworks used: Figma, Miro, Replit, VSCode, Git

Responsibilities: Project lead, UX Research/Design, Prototyping, Development
